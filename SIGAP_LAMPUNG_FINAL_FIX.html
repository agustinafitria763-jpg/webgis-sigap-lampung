🗺️ Jenis Bahaya / Bencana

🌊
Banjir
Kawasan rawan banjir akibat curah hujan tinggi atau luapan sungai. Umumnya terjadi di wilayah pesisir dan dataran rendah.
⛰️
Longsor / Gerakan Tanah
Area berpotensi longsor pada lereng bukit dengan kemiringan >25°. Dipicu curah hujan tinggi dan getaran.
🌍
Zona Gempa Bumi
Area pemantauan aktivitas seismik. Bandar Lampung terletak di zona subduksi aktif Sumatera.
🔥
Kebakaran Lahan/Hutan
Titik rawan kebakaran lahan bervegetasi kering. Umumnya meningkat pada musim kemarau.

Tentang SIGAP-LAMPUNG

SIGAP-LAMPUNG (Sistem Informasi Geospasial Aktif Penanggulangan Bencana – Lampung) adalah platform WebGIS berbasis web yang dirancang untuk memetakan, memantau, dan menganalisis risiko kebencanaan di Kota Bandar Lampung secara real-time.


Sistem ini dikembangkan sebagai bagian dari Tugas Akhir / Karya Ilmiah mahasiswa Program Studi Pendidikan Geografi Universitas Lampung tahun 2026, dengan tujuan mendukung manajemen risiko bencana berbasis spasial.

Fitur Utama

  • Peta interaktif multi-layer dengan toggle layer per jenis bencana
  • Visualisasi data bahaya dari file GeoJSON (hidrometeorologi.geojson)
  • Formulir pelaporan bencana real-time oleh masyarakat
  • Dashboard analisis risiko per kecamatan
  • Basemap berganti: OSM, Satelit, Topografi, Terrain
  • Legenda klasifikasi risiko bencana
  • Monitoring 24 jam dengan indikator live

Cara Penggunaan

  • Login: Masukkan username dan password untuk mengakses dashboard
  • Peta Utama: Klik marker untuk melihat detail titik bahaya
  • Toggle Layer: Aktifkan/nonaktifkan layer bencana di sidebar kiri
  • Basemap: Gunakan kontrol layer kanan atas peta untuk mengganti basemap
  • Lapor Bencana: Klik tombol merah di header untuk melaporkan kejadian
  • Unduh Peta: Gunakan menu Cetak / Unduh Peta di sidebar
  • GeoJSON: Pastikan file hidrometeorologi.geojson ada di folder yang sama dengan file HTML ini

📝 Tentang Proyek

SIGAP-LAMPUNG dikembangkan sebagai karya ilmiah dalam rangka mendukung sistem penanggulangan bencana berbasis geospasial di Kota Bandar Lampung. Proyek ini mengintegrasikan teknologi WebGIS dengan data lapangan untuk menghasilkan peta interaktif yang dapat diakses oleh masyarakat umum dan pemangku kepentingan terkait.


Sistem ini menggunakan framework Leaflet.js untuk visualisasi peta interaktif, dilengkapi dengan data GeoJSON dari hasil survei lapangan, serta fitur pelaporan bencana secara partisipatif oleh masyarakat.
